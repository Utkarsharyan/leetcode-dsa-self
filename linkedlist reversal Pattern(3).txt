=====================================================
LINKED LIST REVERSAL — MASTER PATTERN
=====================================================

PATTERN:
Pointer Reversal

-----------------------------------------------------
CORE IDEA
-----------------------------------------------------

Reverse the direction of next pointers
while traversing the list.

-----------------------------------------------------
GENERIC TEMPLATE
-----------------------------------------------------

Node prev = null
Node curr = head

while(curr != null):

    next = curr.next
    curr.next = prev

    prev = curr
    curr = next

head = prev

-----------------------------------------------------
TIME & SPACE
-----------------------------------------------------

Time: O(n)
Space: O(1)

-----------------------------------------------------
COMMON MISTAKES
-----------------------------------------------------

- Losing reference to next node
- Forgetting to update head
- Breaking the chain accidentally

-----------------------------------------------------
WHEN TO USE
-----------------------------------------------------

- Reverse linked list
- Reverse sublist
- Reverse second half
- Reverse in groups
- Palindrome linked list

LEETCODE QUESTIONS

-----------------------------------------------------
KEY TAKEAWAY
-----------------------------------------------------

Always store next node before
changing pointers.
